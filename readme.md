Third Eye:Smart Assistant for the Blind 


Project Overview::
Third Eye is a project designed to help visually impaired people detect obstacles using sensors.It is currently a prototype in development.


Current Status:: 
.Working:Ultrasonic sensor detection and voice alerts for obstacles are fully functional.
.In Progress:Integrating GPS ,RTC and GSM modules for full voice feedback.
.Debugging:Data for location ,time and temparature currently shows in the Serial Monitor but needs further integration for voice output.


Tech UsedHardware::
 .Arduino, Ultrasonic Sensor (HC-SR04), DFPlayer Mini.
 .Language: C++ (Arduino).


 Goal::
The final goal is to make all sensors talk to the user, providing a complete hands-free navigation assistant.
